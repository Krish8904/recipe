# Recipe Book App

## Description

The Recipe Book App is a web application that allows users to browse, search, and add recipes. It features a collection of preloaded Indian recipes and supports adding custom recipes with details including ingredients, preparation steps, and images.

## Features

* Browse preloaded Indian recipes in a card layout.
* Add new recipes with recipe name, ingredients, steps, and an image.
* Search recipes by name or ingredient.
* Click on a recipe card to view full details.
* Responsive design for desktops and mobile devices.

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Clone or download the repository.
2. Open the `index.html` file in a web browser.
3. Browse existing recipes or add a new recipe by filling out the form.
4. Use the search bar to find recipes by keyword or ingredient.
5. Click on a recipe card to view its details.

## Validation

* Ensures that all input fields are filled before adding a new recipe.
* Displays error messages if required information is missing.
